Figures created from a matplotlib plot will be rendered below the code cell:

import matplotlib.pyplot as plt plt.plot([0,1], [1,0])

There is no need to use plt.show().

Multiple Figures

If multiple figures are created, they will all be rendered:

import matplotlib.pyplot as plt fig1, ax1 = plt.subplots() ax1.plot([0,1], [1,0]) fig2, ax2 = plt.subplots() ax2.plot([0,1], [1,0])

Related Libraries

Figures created from other plotting libraries based on matplotlib (e.g. pandas, seaborn) will also be rendered:

import pandas as pd import matplotlib.pyplot as plt df = pd.DataFrame({ 'x': [1,2,3,4,5], 'y': [5,4,3,2,1] }) df.plot(x='x', y='y', kind='line') import seaborn as sns x = [1,2,3,4,5] y = [5,,0, 5, 0, 5] sns.scatterplot(x=x, y=y)
